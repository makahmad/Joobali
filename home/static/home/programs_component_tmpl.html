Programs
  A program is a set dollar amount to bill parents on a weekly or monthly basis. Some common program names include Full-time, Part-time, and Extended care. You can have an unlimited number of programs. You can add multiple programs to a child. For example, a parent may be billed for both the Full-time and Extended care programs on one invoice.
